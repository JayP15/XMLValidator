# XML Validator and Converter

This project provides a Java application that can:
1. Convert a JSON file to an XML file.
2. Validate an XML file against an XSD schema.

The main functionality is encapsulated in the `XMLValidator` class, which offers methods for both converting JSON to XML and validating XML files against XSD.

## Features
- **JSON to XML Conversion**: Converts a JSON file into XML format while handling a simple root structure.
- **XML Validation**: Validates an XML file against a provided XSD schema, checking for correctness according to the defined rules.

## Requirements
- Java 8 or higher
- Jackson Library (for JSON and XML processing)
- Xerces (for XML schema validation)

## Setup

### Prerequisites
- Download and install Java 8 or higher.
- Add dependencies for Jackson and Xerces libraries. You can use Maven or Gradle for dependency management.
